# 光遇今日国服插件

这是一个为 AstrBot 框架移植的光遇今日国服攻略查询插件，基于原 nonebot_plugin_sky 的核心功能。

## 功能

- 查询光遇国服每日任务攻略
- 获取微博@今天游离翻车了吗 @陈陈努力不鸽 的最新攻略信息
- 支持文本和图片内容展示（自动保存图片到临时目录）
- 智能清理临时文件，避免磁盘空间占用
- **自动推送功能**：支持每日定时推送攻略到指定群组或好友
- **智能缓存系统**：数据缓存3小时，避免频繁请求被拉黑

## 使用方法

### 手动查询
发送以下任一命令即可查询今日国服攻略：

- `/今日国服`

### 推送设置帮助
发送以下命令获取自动推送配置帮助：

- `/光遇推送设置`

### 清除缓存
发送以下命令清除缓存在本地的数据：

- `/清除缓存`

## 安装

1. 将文件放入 AstrBot 的插件目录
3. 安装依赖：`pip install -r requirements.txt`
4. 重启 AstrBot 或在 WebUI 中重载插件
